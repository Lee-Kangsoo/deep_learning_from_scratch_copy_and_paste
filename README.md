# Deep Learning From Scratch

책을 읽으면서 직접 구현한 예제 코드와 실습 내용을 정리하는 저장소입니다.

## Overview

이 프로젝트는 다음 흐름으로 공부합니다.

- 책 내용을 직접 손으로 구현하기
- 핵심 개념을 코드로 다시 확인하기
- 실습 단위를 작게 나누어 commit 남기기

## Tech Stack

- Python
- NumPy
- Matplotlib

## Goal

코드를 "읽는 것"에서 끝내지 않고, 직접 작성하면서 이해를 쌓는 것을 목표로 합니다.

## Project Structure

실습이 진행되면서 아래처럼 정리해 나갈 예정입니다.

```text
deep_learning_from_scratch/
├── pyproject.toml
├── README.md
└── chapters/
```

## Getting Started

의존성 설치:

```bash
pip install -e .
```

예제 실행:

```bash
python path/to/example.py
```
